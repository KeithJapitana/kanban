# Kanban MVP

Single-board Kanban app with drag and drop, renameable columns, add/delete card actions, dummy data, and no persistence.

## Requirements

- Node.js 18 or higher

## Commands

```bash
npm install
npm run dev
npm run lint
npm run test:ci
npm run test:e2e
npm run build
npm start
```

## Production Deployment

```bash
npm install
npm run build
npm start
```

App runs at http://localhost:3000
